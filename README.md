# Sales Data Analysis using Python
# Objective

The objective of this project is to perform exploratory data analysis (EDA) on a structured sales dataset to identify trends, patterns, and key business insights.

---

## Tools & Technologies

* Python
* pandas

---

## Dataset

* Superstore Sales Dataset (CSV format)
* Contains information about orders, customers, products, sales, profit, and regions

---

## Data Preprocessing

* Removed duplicate records
* Handled missing values using forward fill
* Converted `sales` column to numeric format
* Parsed date columns (`order_date`, `ship_date`) using proper datetime handling

---

## Analysis Performed

### Total Sales

* Calculated overall revenue generated

### Sales by Category

* Identified top-performing product categories

### Sales by Region

* Analyzed region-wise performance

### Monthly Sales Trend

* Evaluated sales trends over time

### Top Products

* Identified highest revenue-generating products

### Profit Analysis

* Compared profit across categories

### Loss Analysis

* Detected transactions with negative profit

---

## Key Insights

* Certain categories contribute significantly higher revenue
* Regional performance varies, indicating market differences
* Some transactions result in losses despite high sales
